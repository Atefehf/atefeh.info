# Magnetic Button
 _A Pen created at CodePen.io. Original URL: [https://codepen.io/nw/pen/eGoGRe](https://codepen.io/nw/pen/eGoGRe).

 I really like the button hover interaction on https://simplychocolate.dk, so wanted to have a play with creating something kinda similar.

Using Rebound.js (http://facebook.github.io/rebound-js/docs/rebound.html) for the springiness and relatively straight-forward math + event listeners + styling to tie it together.

Probably desktop-only, since it's a hover effect.
